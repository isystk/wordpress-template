## Usage

### DockerWindows(WSL)を利用している場合は以下の設定が必要です。
$ vi ~/.bashrc
``` 
export DOCKER_HOST=tcp://localhost:2375
```

$ sudo vi /etc/wsl.conf
``` 
[automount]
root = /
options = "metadata"
```

### 使い方を見る場合
``` bash
$ ./dc.sh -h
```

## Install

vi ./.env
```
PROJECT_NAME=<利用するプロジェクト名>
```
